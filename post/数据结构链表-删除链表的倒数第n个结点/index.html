[数据结构]链表 删除链表的倒数第N个结点

Jan 25, 2020
1 min read

题目描述

给定一个链表，删除链表倒数第n个结点，并返回链表的头结点。

示例：
给定一个链表: 1->2->3->4->5, 和 n = 2.
当删除了倒数第二个节点后，链表变为 1->2->3->5.

算法：

解这个题目时，比较容易想到的是遍历两次链表，第一次遍历找出倒数第n个结点(即正数第L-n+1个结点)，第二次遍历让其前一个结点的指针指向下下一个结点。难在尝试用一次遍历来完成整个过程，方法：定义两个指针，两个指针间隔为n，保持这种间隔移动两个指针，当前一个指针为null时，则后一个指针指向的则是要删除结点的前一个结点，直接修改其next即可。

题解：

class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode cur = new ListNode(0);

        cur.next = head;
        ListNode p = cur;
        ListNode q = cur;
        int flag = 0;
        //让两个指针间隔为n
        while (flag != n+1)
        {
            q = q.next;
            flag++;
        }
        //移动两个指针直到前一个指针为null
        while (q != null)
        {
            p = p.next;
            q = q.next;
        }
        p.next = p.next.next;
        return cur.next;
    }
}

复杂度：

  • 时间复杂度：假设链表长度为n，则为O(n)
  • 空间复杂度：O(1)
