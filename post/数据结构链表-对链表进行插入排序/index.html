[数据结构]链表–对链表进行插入排序

题目描述

对链表进行插入排序

示例1：
输入: 4->2->1->3
输出: 1->2->3->4
示例2：
输入: -1->5->3->4->0
输出: -1->0->3->4->5

插入排序算法：

1.插入排序是迭代的，每次只移动一个元素，直到所有元素可以形成一个有序的输出链表。

2.每次迭代中，插入排序只从输入数据中移除一个待排序的元素，找到它在序列中适当的位置，并将其插入。

3.重复直到所有输入数据插入完为止。

题解：

class Solution {
    public ListNode insertionSortList(ListNode head) {
        if (head == null || head.next == null)
            return head;
        ListNode sentinel = new ListNode(-1);
        sentinel.next = head;
        ListNode pre = sentinel;
        ListNode cur = head;
        ListNode last;

        while (cur != null)
        {
            last = cur.next;
            if (last != null && cur.val > last.val)
            {
                while (pre.next != null && pre.next.val < last.val)
                {
                    pre = pre.next;
                }
                cur.next = last.next;
                last.next = pre.next;
                pre.next = last;

                //pre复位
                pre = sentinel;
            }
            else{
                cur = last;
            }
        }
        return sentinel.next;
    }
}

复杂度：

  • 时间复杂度：O(n²)
  • 空间复杂度：O(1)
