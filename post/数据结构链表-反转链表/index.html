[数据结构]链表–反转链表

题目描述

反转一个单链表

示例：
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL

算法1(循环)：

遍历链表，将当前结点的next指向当前结点的前一个结点，由于结点定义时没有定义前驱指针，故必须提前保存当前结点的前一个结点，在进行更改之前，还需保存当前结点的下一个结点。

题解：

class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;
        ListNode current = head;
        ListNode next = null;
        while (current != null)
        {
            next = current.next;
            current.next = pre;
            pre = current;
            current = next;
        }
        return pre;
    }
}

复杂度：

  • 时间复杂度：假设链表长度为n，则为O(n)
  • 空间复杂度：O(1)

算法2(递归)：

假设链表是[1, 2, 3, 4, 5]从最底层最后一个reverseList(5)来看

  1. 返回了5这个节点
  2. reverseList(4)中
  3. p为5
  4. head.next.next = head 相当于 5 -> 4
  5. 现在节点情况为 4 -> 5 -> 4
  6. head.next = null,切断4 -> 5 这一条，现在只有 5 -> 4
  7. 返回（return）p为5，5 -> 4
  8. 返回上一层reverseList(3)
  9. 处理完后返回的是4 -> 3
  10. 依次向上

题解：

class Solution {
   public ListNode reverseList(ListNode head) {
    if (head == null || head.next == null)
        return head;
    ListNode p = reverseList(head.next);
    head.next.next = head;
    head.next = null;
    return p;
}
}

复杂度：

  • 时间复杂度：假设链表长度为n，则为O(n)
  • 空间复杂度：O(n)，由于使用递归，将会使用隐式栈空间，递归深度可能达到n层。
